JavaScript

Otchy がちょこちょこと書いた、JavaScript のライブラリとかを公開するページ。
思い立ったら吉日とばかりに、勢いだけで作るようなものを置く予定。
  • TwitAPI.js
    Twitter API を JavaScript から簡単に使うためのライブラリ。
    専用のプロキシを利用し、クロスドメインの問題や PUT/DELETE メソッドへの対応を解決。
  • jquery.tablefix.js
    Excel のウィンドウ枠固定のように、テーブルのヘッダや左側を固定して、残りの部分をスクロールさせるための、jQuery プラグイン。
  • jquery.jaticker.js
    日本語を変換しながら入力しているように文字列をアニメーションさせる jQuery プラグイン。
  • jquery.smoothswap.js
    ぬるっと動いて、2つのパネルの位置が入れ替わる jQuery プラグイン。
  • コナミコマンド.js
    ウェブページに簡単にコナミコマンドを追加するためのスクリプト。
  • MultiLink.js
    1つのリンクで複数サイトにリンクさせるためのスクリプト。
