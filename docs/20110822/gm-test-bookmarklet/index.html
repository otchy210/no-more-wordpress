起動すると、user.js の URL を聞いてくるので、入力すると今開いているページにそのスクリプトを読み込みます。
単にその user.js をそのページに追加するだけの単純な仕組みなので、GM_* 等への対応はありません。

任意のサイトを対象とした user.js を書くにあたって、自分のテストを楽にするために書きました。
