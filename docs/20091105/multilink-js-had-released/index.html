1つのリンクで複数サイトにリンク出来る MultiLink.js 公開

１つのリンクで複数サイトにリンクできる便利JavaScriptライブラリ「Pluralink」 で紹介されていた、Pluralink というライブラリですが、JavaScript 無効時の事は考慮されていないし、必要なファイルがいっぱいあって面倒くさいし、マークアップが HTML 的にアレだったりしたので、その辺を解決したライブラリを作ってみました。

こう書くと、
いろんな<span class="multi-link" title="検索エンジン">検索エンジン(
    <a href="http://www.google.com">Google</a>・
    <a href="http://www.yahoo.com">Yahoo!</a>・
    <a href="http://www.bing.com">bing</a>
)</span>があるから好きなのを使うといいよ！

こうなります。

いろんな検索エンジン( Google・ Yahoo!・ bing )があるから好きなのを使うといいよ！

気になったら是非！
