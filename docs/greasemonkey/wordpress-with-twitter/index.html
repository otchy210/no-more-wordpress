WordPress with Twitter

»Install


WordPress に記事を投稿すると同時に、Twitter にも投稿するためのスクリプトです。
管理画面のインターフェースが大幅に変更された、WordPress 2.7 に対応しています。
以前のバージョンでは動作確認していませんが、全く動かない事請け合いです。

スクリプトをインストールすると、ページや投稿の編集・公開時に下の画面のような withT チェックボックスが追加されます。



このチェックボックスにチェックを付けてから編集や公開を行うと、その内容を Twitter に投稿する事が出来ます。
ブラウザ起動後、初回実行時は Twitter の認証のためのダイアログがポップアップするので、認証情報を入力して下さい。

2009-01-06


公開日時を編集したり、過去に作成したエントリを新規公開した場合にパーマリンクが確定していないため、誤った URL にリンクしてしまう事がある不具合に対応。 (警告を出す)
パーマリンクが長すぎて省略されている場合に、誤った URL にリンクしてしまう不具合に対応。
