Pure JavaScript 実装で、Twitter API を使うライブラリ、TwitterAPI.js のバージョン 0.9.2 をリリースしました。

今回、少し不具合を直したというのもあるのですが、従来になかった大きな違いとして、既存の Twitter API にない、新しいメソッドを追加したというのがあります。
TwitterAPI.users.own がそれで、認証されたユーザ自身の情報を取得するメソッドです。
元々あっても良さそうなメソッドなのですが無かったので追加しました。

TwitterAPI.js については、新しい方向付けとして、元々の Twitter API の上位互換 (？) 的な位置付けはありだなぁ。と思う今日この頃です。

あるいは将来的に、Twitter API 側の仕様が変更されて、従来通りの書き方が通用しなくなった場合 (過去の Twitter API の仕様変更ではそういった事が多かったようです) などに、仕様の差異を TwitterAPI.js で吸収して、ライブラリを利用しているアプリケーションでは何も修正しなくて良いというようなのも、今後の展開としてはありですね。
