htn.to からコメントページに飛ばされてしまったとき元ページにリダイレクトするユーザースクリプト

はてなブックマークの仕様改悪がそっと始まったので、みんなで設定変更しよう（懇願） なんていう話もありますが、ブックマークをする側が設定変更したところで、デフォルトのままの人が多ければ、結局いったんコメントページを踏まされてしまう事になるので、自衛のためのユーザースクリプトを書きました。

htn.to 経由ではてブコメントページに飛ばされたとき、自動的に元ページにリダイレクトします。
HTML 内に Tweet ブロックが存在するかどうかで判別しているので、はてな側の仕様変更で動かなくなる可能性はあります。
