現在こちらのスクリプトは動作していません。代わりに作成した Chrome エクステンションをご利用下さい。

Google Reader の終了に先だって Feedly へ移行したのですが、Feedly はやはり海外製だけあってはてなブックマーク数の表示などには対応していないので、それに対応させるユーザースクリプトを書きました。

Google Chrome の場合、以下のリンクから user.js ファイルをダウンロードして、ツール＞拡張機能 のところにドロップすると、インストールすることが出来ます。
そのままクリックしただけではインストール出来ないのでご注意下さい。

Download (Ver. 0.7.2) -- New! Feedly のフィードが feedly.com ドメインで提供されるようになった変更に対応。

(2013-07-24 追記) Ver 0.7.0 で一覧でのブクマ数表示に対応しましたが、Google Reader 移行組なのでリストビュー以外は使っておらず、例えばカードビュー(？)みたいなところは未対応です。要望あれば検討します。

(2013-06-26 追記) 昨日、今日あたりで正常動作しなくなっているようです。一瞬表示されることもあるので、Feedly の HTML 構造が変更されたりしたのが原因ではないかと思います。Firefox 版は動いているので、Chrome の自動更新が原因と思われます。
(自分が不便なので) 近日中に修正版を作ってアップします。

(2013-07-03 追記) 単に適用ドメインの問題で、www.feedly.com から cloud.freedly.com にリダイレクトされていたのが原因でした。自分のローカルが Ver 0.5.0 のままだったのを忘れていたという情けない原因で、Ver 0.5.1 にあげたらあっさりなおったので、特に修正版は作っていません。

一覧でのブクマ数表示の要望が多いので、それはそのうち対応しようかなと思ってます。あと保守が面倒なので、Chrome / Fx コードの共通化が課題ですね。
